# 社区行为准则

本项目采用 Contributor Covenant 2.1 的核心原则，希望所有参与者都能在开放、友善、无骚扰的环境中贡献中文 AI 教程。

## 我们的承诺

无论年龄、外貌、身体状况、民族、性别认同与表达、经验水平、教育背景、社会经济状况、国籍、个人形象、种族、宗教或性取向如何，项目维护者和贡献者都承诺为每个人提供安全、包容的参与体验。

## 鼓励的行为

- 对不同经验水平保持耐心，解释结论时提供可核验依据；
- 尊重不同观点、使用场景和技术选择；
- 接受建设性反馈，并对自己的错误负责；
- 关注对社区整体最有帮助的结果；
- 在公开内容中保护他人的隐私、凭据和账户信息。

## 不可接受的行为

- 侮辱、歧视、威胁、跟踪、性骚扰或其他针对个人的攻击；
- 发布他人的私人信息、API Key、Cookie、日志、联系方式或账户数据；
- 故意传播已知错误、伪造使用经历或操纵性推广内容；
- 在 Issue、Pull Request 或讨论中反复发布垃圾信息；
- 维护者认为不适合专业协作环境的其他行为。

## 执行责任

维护者负责解释并执行本准则，可以删除、编辑或拒绝违反准则的评论、提交、代码、Issue 和其他贡献。执行时会尽量说明理由，并根据影响范围采取提醒、临时限制或永久禁止参与等措施。

## 适用范围

本准则适用于仓库、Issue、Pull Request、与项目相关的公开交流空间，以及代表本项目参与其他社区活动时的行为。

## 私密报告

请通过 GitHub 的[私密安全报告](https://github.com/18534516725/llm-api-setup-guides/security/advisories/new)报告严重骚扰、隐私泄露或安全问题。维护者会尽快审查，并在处理过程中尊重报告者的隐私。

## 来源

本准则基于 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 的原则编写。
